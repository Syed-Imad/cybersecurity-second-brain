## The Learning Method

Each module follows a 3-phase cycle:

| Phase | What happens |
|-------|-------------|
| **1 — Ingestion** | Claude reads all raw materials and teaches the content using analogies, visuals, and examples |
| **2 — Active Recall** | Close everything and write what you remember from scratch. Find the gaps. |
| **3 — Testing** | Scenario-based questions. Write full answers. Get feedback. |

Then every concept becomes an atomic Zettelkasten note in Obsidian, linked to related ideas across all modules.

## How to Use This With Claude

Start any session by saying:

```
"Read CLAUDE.md and let's work on [Module Name]"
```

Claude will read `CLAUDE.md`, pick up where you left off, and teach the content in your exact learning style — analogies first, fundamentals before detail, visualisation throughout.
