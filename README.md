# Cybersecurity Second Brain

> A personal knowledge base for building deep, lasting understanding of cybersecurity across 3 years of university, work experience, and self-study.
> Built with Obsidian + Zettelkasten. Ingested and generated by Claude.

---

## How This System Works

```
Raw Materials/          ← Drop source files here (slides, PDFs, ZIPs)
       ↓
   Claude reads and processes
       ↓
Knowledge Base/         ← Distilled Zettelkasten notes live here
                           (what you learn from, query, and feed to AI)
```

**You never read raw slides.** Claude ingests them and produces atomic, linked, analogy-rich notes that go directly into the Knowledge Base.

## To Start a New Module
1. Drop raw files into `Raw Materials/Year X/Module Y/`
2. Open Claude Code in this folder
3. Say: `"Read CLAUDE.md and ingest [module name]"`
4. Claude reads everything and populates `Knowledge Base/Concepts/`
