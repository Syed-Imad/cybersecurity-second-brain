# Work & Internships

A log of professional experience, skills gained, and knowledge built outside of university.

## Active / Upcoming

> Add internship or job details here when confirmed.

## Log Template

When starting a new role, create a subfolder: `Company Name - Role - Year/`

Each role folder should contain:
- `Overview.md` — company, role, dates, team, tech stack
- `Skills-Learned.md` — technical and soft skills gained
- `Projects.md` — work done, problems solved, impact
- `Reflections.md` — what went well, what was hard, what you'd do differently
